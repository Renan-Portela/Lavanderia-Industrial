# Quickstart: Hybrid Tracking

## Setup
1. Ensure `MaterialService` has at least one SKU category (e.g., `GLV-IND-XL-A`).
2. Navigate to `/pages/recebimento.php`.

## Testing the Hybrid Flow
1. Select a SKU from the catalog dropdown.
2. Type a specific description in the "Descrição do Material" field.
3. Save and check the generated QR detail.
4. Scan the QR in `/pages/lavagem.php` and verify both SKU and Description are shown.
